HTMLSky

An HTML-only Bluesky frontend built by @jordanreger.com.

Seriously... There's no JavaScript anywhere on this site. Nice, huh?

Why?

It's nice! There's no downsides to not having JavaScript. Plus, it loads quite a lot faster thanks to Server-Side Rendering (SSR).

Something's missing!

Tell me! As of 2024-02-19, this project is still in the works, so chances are you will find something.
